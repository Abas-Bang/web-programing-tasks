House Price Prediction with Linear Regression

Predicted house prices using linear regression, Python, and Tableau

Details This project aimed to predict house prices based on various features such as square footage, number of bedrooms, and location. We employed linear regression techniques in Python, utilizing libraries like NumPy and scikit-learn to build predictive models. The project also involved data visualization using Tableau to present insights into house price trends and factors influencing the market. The outcomes included accurate price predictions and data-driven recommendations for homebuyers and sellers.
